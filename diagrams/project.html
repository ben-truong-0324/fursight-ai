FurrSight AI Project Phases

Transforming dog grooming insights through intelligent image processing

Phase 1: Query Generation

Leveraging a Large Language Model (LLM) to generate diverse search queries related to dog grooming.

Phase 2: Web Scraping

Deploying an automated scraper to gather URLs for images, videos, and web content from platforms like Google Images and YouTube.

Phase 3: Data Extraction & Structuring

Processing scraped content to identify and download relevant media, focusing on "before" and "after" image pairs. Using a Vision-Language Model (VLM) to analyze images and text, then organizing the data into a clean, structured format.
